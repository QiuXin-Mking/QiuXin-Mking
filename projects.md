# Projects

## Self-Improvement ⭐ 36

**Spaced Repetition Learning System** — Go + Vue 3 · CLI + Web

A scientific learning system based on the Ebbinghaus Forgetting Curve. Smart algorithm
automatically schedules review times to maximize knowledge retention.

### Key Features
- 🖥️ **Dual Interface**: CLI (`cli_server.go`) + Web (Go backend + Vue 3 frontend)
- 🧠 **Smart Scheduling**: 4-level feedback system with adaptive interval multipliers
- 🔐 **Multi-tenant**: JWT auth, bcrypt password hashing, SQLite storage
- 📱 **Mobile-First**: Vant UI components, responsive design
- 📝 **Markdown Parser**: Auto-parse `.md` knowledge bases into questions

### Tech Highlights
- Backend: Gin + GORM + JWT (HS256)
- Frontend: Vue 3 + TypeScript + Pinia + Axios
- Build: Vite + Go cross-compilation
- Deploy: Shell scripts + systemd
